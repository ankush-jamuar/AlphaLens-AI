# Session 01 — Project Planning

## Goal

Understand the assignment requirements and design a scalable AI-powered investment research platform.

---

## Topics Covered

- Requirement analysis
- Technology stack selection
- Folder structure
- Product scope
- Architecture brainstorming
- UI planning

---

## Major Decisions

- Use Next.js App Router.
- Use LangGraph for workflow orchestration.
- Use Prisma ORM with Neon PostgreSQL.
- Use Clerk for authentication.
- Use Gemini for investment reasoning.
- Use Alpha Vantage for financial data.
- Use GNews for market news.

---

## Outcome

Defined the overall system architecture and implementation roadmap for AlphaLens AI.
